## 如何貢獻

歡迎所有形式的貢獻來完善這個術語對照表！

#### 1. 增加新的術語對照
- 在 `terms.csv` 文件中增加新的術語對照
- 格式：`中國大陸術語,台灣術語`
- 請確保術語的準確性和適用性

#### 2. 回報問題
- 如果發現現有術語對照有誤，請發出 Issue
- 描述問題並提供正確的對照建議

### 貢獻原則

- **準確性優先**：確保術語對照的準確性
- **上下文相關**：考慮術語在不同上下文中的使用
- **一致性**：保持與現有對照表的一致性
- **可讀性**：確保添加的內容清晰易懂
